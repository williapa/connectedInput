Connected Input

Connected input marries a custom slider with a validated text input, formatted for currency use. CSS offers the cross-browser slider styles, while javascript offers following features:

  • Min and max set through html attributes
  • Value for both fields always connected, one formatted string & one digit-only
  • Text field stripped of all non-digits and reformatted on blur
  • Step value set through html to prevent ugly decimals
  • Gradient toggled through html attribute
  • Dollar sign and commas formatted through javascript
  • Best of all? No need for dumb, tacky validation errors!
  • ADA compliant
